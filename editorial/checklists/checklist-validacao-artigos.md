# Checklist de validacao de artigos sobre cuidados capilares (publico leigo)

Use este checklist na revisao final antes de publicar.

## SEO on-page
- [ ] Titulo (H1) simples e direto, com a palavra-chave principal.
- [ ] Primeiros 100-150 caracteres dizem o tema em linguagem clara.
- [ ] Estrutura com H2/H3 organizada por topicos (sem pular niveis).
- [ ] Palavra-chave e variacoes aparecem de forma natural.
- [ ] Links internos para artigos relacionados e uteis para iniciantes.
- [ ] Links externos apenas para fontes confiaveis e simples.
- [ ] Imagens ajudam a entender e tem texto alternativo claro.

## Clareza do conteudo
- [ ] O texto diz para quem e (ex.: cabelo seco, cacheado, oleoso).
- [ ] O objetivo aparece logo no inicio.
- [ ] Passos em ordem, com frequencia e tempo quando necessario.
- [ ] Termos tecnicos explicados com exemplos do dia a dia.
- [ ] Nao ha frases vagas como "use quando precisar".
- [ ] Conclusao resume o que fazer e o proximo passo.

## Linguagem humanizada
- [ ] Tom amigavel, falando com "voce".
- [ ] Exemplos praticos de rotina (ex.: como aplicar a mascara).
- [ ] Evita jargao e linguagem de especialista.
- [ ] Frases curtas, leitura leve.
- [ ] Nao parece texto automatico ou impessoal.

## Conformidade com Google AdSense
- [ ] Nao promete resultado garantido.
- [ ] Nao faz afirmacoes medicas ou de cura.
- [ ] Se houver irritacao ou problema serio, recomenda procurar profissional.
- [ ] Nao incentiva misturas caseiras perigosas.
- [ ] Linguagem respeitosa e segura.

## Estrutura em Markdown
- [ ] Apenas um H1 no inicio do texto.
- [ ] H2 para secoes principais e H3 para subtitulos.
- [ ] Listas com "-" ou numeracao para passos.
- [ ] Links no formato `[texto](url)`.
- [ ] Imagens no formato `![alt](url)` quando usadas.
- [ ] Sem HTML desnecessario.

## Aprovacao final
- [ ] Revisao ortografica e de coerencia concluida.
- [ ] Checklist 100% marcado.
